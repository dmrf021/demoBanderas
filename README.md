# demoBanderas
Esta es una demo de un juego de relacionar banderas con su pais/region.

## Requisitos

- Python 3.10 o superior
- pip

## Instalación

```bash
git clone https://github.com/usuario/demoBanderas.git
cd demoBanderas
pip install cairo cairosvg pygame
```
## Estructura esperada

```
/demoBanderas
├── main.py
├── README.md
├── data
│   └── flags.json
├── assets
│   ├── flags
│     └── flags.svg
└── scripts
    ├── descargar_banderas.py
    └── create_json.py

```


## Preparación

- Ejecuta **descargar_banderas.py** y **crear_json.py**
- Ejecuta **main.py**
